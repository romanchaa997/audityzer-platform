# Frontend

React + TypeScript frontend for Audityzer Platform.

## Features
- MetaMask integration
- Hybrid Post-Quantum signer (liboqs-node)
- Smart contract audit dashboard
- Real-time security monitoring

## Tech Stack
- React 18
- TypeScript
- ethers.js / web3.js
- liboqs-node for PQ cryptography
- TailwindCSS

## Getting Started

```bash
cd frontend
npm install
npm run dev
```
